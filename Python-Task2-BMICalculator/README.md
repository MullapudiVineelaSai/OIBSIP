# Task 2: BMI Calculator
### Oasis Infobyte Internship Program

## Project Description:
This is a simple Python program that calculates Body Mass Index (BMI). 
BMI helps us understand if a person's weight is healthy according to their height.

## Features:
- Takes height and weight as input from the user
- Calculates BMI using the standard formula
- Shows the BMI category: Underweight, Normal, Overweight, or Obese
- Easy to use for beginners

## BMI Formula:
BMI = weight(kg) / (height(m) * height(m))

## BMI Categories:
- Below 18.5  = Underweight
- 18.5 - 24.9 = Normal weight
- 25.0 - 29.9 = Overweight
- 30.0 and above = Obese
