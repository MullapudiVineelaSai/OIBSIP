# Task 4: Basic Weather App - Beginner Tier
### Oasis Infobyte Internship Program

## Project Description:
This is a Basic Weather App built using Python. 
User enters a city name and the app fetches real-time weather data using OpenWeatherMap API and displays temperature, humidity and weather condition.

## Features:
- City Name Input: User can enter any city name
- Real-time Weather Data: Fetches data from OpenWeatherMap API
- Displays: Temperature in Celsius, Humidity, Weather Description
- Error Handling: Shows message if city not found or internet error
- Simple and Beginner Friendly Code

## How to Run:
1. Install Python 3.x
2. Install required library: pip install requests
3. Get free API key from: https://openweathermap.org/api
4. Paste your API key in the code at API_KEY variable
5. Open Command Prompt / Terminal
6. Go to folder:PS C:\Users\mulla\OneDrive\Desktop\Oasis-Internship\Python-Task4-WeatherApp> 

7. Run: python Task4.py
8. Enter city name when asked

## Technologies Used:
- Language: Python 3.x
- Library: requests
- API: OpenWeatherMap API
