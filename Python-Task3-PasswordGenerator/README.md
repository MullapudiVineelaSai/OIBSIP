# Task 3: Random Password Generator - Advanced GUI
### Oasis Infobyte Internship Program

## Project Description:
This is an Advanced GUI-based Random Password Generator built using Python and Tkinter. 
Users can generate strong passwords by selecting length and character types through a user-friendly interface.

## Features:
- Password Length Slider: Range from 8 to 32 characters. Default: 12
- Character Type Selection: 4 checkboxes
    - Include Uppercase A-Z
    - Include Lowercase a-z  
    - Include Numbers 0-9
    - Include Symbols !@#$%^&*
- Generate Password Button: Creates random password instantly
- Copy to Clipboard Button: Copies password with 1 click
- Error Handling: Shows warning if no character type selected or password not generated
- Clean GUI: Fixed window size 400x380, non-resizable

## How to Run:
1. Install Python 3.x
2. Install required library: pip install pyperclip
3. Open Command Prompt / Terminal
4. Go to folder:PS C:\Users\mulla\OneDrive\Desktop\Oasis-Internship\Python-Task3-RandomPasswordGenerator> 


5. Run: python Task3.py
6. Set length, select checkboxes, click "Generate Password"
