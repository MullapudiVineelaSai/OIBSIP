# Voice Assistant - Task 1

## Description:
A Python-based voice assistant that takes voice commands and responds using text-to-speech. It can tell time, open websites, and answer basic queries.

## Technologies Used
- Python
- speech_recognition
- pyttsx3

## Requirements to Install:
pip install speechrecognition pyttsx3 pyaudio
